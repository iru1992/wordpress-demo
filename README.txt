Wordpress full site editing theme named 'homey' and gutenburg block plugins.

Installation theme

1. Open a Wordpress project and move the 'homey' theme folder into the /wp-content/themes/ folder.
2. Activate the theme via the admin dashboard or WP-cli.

Installation plugins

1. Open a Wordpress project and move the individual plugin folders into the /wp-content/plugins/ folder.
2. Open the plugin folder in the command line and run the npm install command.
3. Run the npm run build or npm start command to generate a build.
4. Activate the plugins via the admin dashboard or WP-cli.
